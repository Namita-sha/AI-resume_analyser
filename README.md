# AI Resume Analyzer

AI-powered Resume Analyzer built with React Router v7, TypeScript, and Puter.js.

Users can upload a resume, receive an AI-generated score, and view structured feedback including strengths and improvement suggestions.

---

## Features

- Resume upload (PDF)
- AI-generated resume score (0–100)
- Strengths & weaknesses analysis
- Resume deletion
- Serverless authentication & storage (Puter.js)

---

## Tech Stack

- React Router v7 (Framework Mode)
- TypeScript
- Puter.js (Auth + Storage + AI)
- Vite
- Tailwind CSS

---

## Local Setup

```bash
npm install
npm run dev
```
